AI debate card cutter

Cut source text into one clean debate card at a time.

Use the backend path for local-model cutting, or use ChatGPT manually without API calls. The app follows the file pattern: tag line, cite line, then evidence paragraph, with a validation pass before copy.

1 Tag

Enter the rough debate claim you want to cut.

2 Research

Find the most likely source and fill the evidence form.

3 Cut

Return one formatted card with cite line, read text, and context.

4 Validate

Check whether the card is actually useful, then revise it if needed.

5 Copy

Copy the final debate-ready card block.

Choose a workflow

Local model via backend, or manual ChatGPT copy/paste and validation.

ChatGPT subscription still is not an official app API. Use the manual workflow for the supported path, or point the backend at an experimental external bridge if you choose to run one.
